# HashProbe Web 🌐

A modern web dashboard for HashProbe analysis. 

> [!IMPORTANT]
> **Status**: Under Development 🛠️

## 🎯 Planned Features

- **Drag & Drop**: Upload files containing multiple hashes.
- **Visual Reports**: Charts and graphs of analysis results.
- **Real-time Cracking**: Progress bars and live updates.
- **API Access**: Integration for other tools.

## 🏗 Tech Stack (Planned)

- **Frontend**: Next.js / Tailwind CSS
- **Backend**: FastAPI / Flask
- **Database**: PostgreSQL (for history)
